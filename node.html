NODE

NODE es un sistema de mobiliario modular diseñado para adaptarse a diferentes configuraciones espaciales y necesidades de uso, a partir de una lógica estructural común.

El taburete NODE aplica la misma lógica constructiva a un asiento individual, manteniendo la identidad del sistema y garantizando ergonomía, ligereza visual y robustez estructural.

El proyecto se apoya en planos constructivos que definen las uniones, materiales y tolerancias necesarias para una producción industrial viable, demostrando la relación entre diseño y viabilidad productiva.

NODE muestra cómo cada pieza se ensambla mediante una lógica clara y eficiente, con el uso de materiales sencillos y un diseño funcional que busca la máxima versatilidad.
