NODE

NODE es un sistema de mobiliario modular concebido para adaptarse a diferentes configuraciones espaciales y necesidades de uso, a partir de una misma lógica estructural y construct l.

La versión sofá del sistema NODE traduce los principios modulares en una pieza de descanso confortable, manteniendo coherencia formal, estabilidad estructural y una clara vocación de uso doméstico y contract.

El taburete NODE aplica la misma lógica constructiva a un asiento individual, conservando la identidad del sistema y garantizando ergonomía, ligereza visual y robustez estructural.

El desarrollo técnico del proyecto se apoya en planos constructivos que definen con precisión las uniones, materiales y tolerancias necesarias para una producción industrial viable.

Estos planos reflejan la relación entre diseño formal y viabilidad productiva, mostrando cómo el sistema NODE se concibe desde una lógica clara de ensamblaje y fabricación.
