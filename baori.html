BAORI

BAORI es un sistema de mobiliario modular diseñado para espacios de uso colectivo. El proyecto busca fomentar la interacción social a través de una configuración flexible, adaptable a diferentes contextos y necesidades.

El sistema se compone de distintos módulos que permiten múltiples disposiciones, integrando superficies de apoyo, almacenamiento y elementos auxiliares. La estructura facilita tanto el uso individual como colectivo.
