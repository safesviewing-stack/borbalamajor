BAORI

BAORI is a modular furniture system designed for collective and public spaces. The project focuses on flexibility, adaptability, and efficient use of space, allowing multiple configurations depending on user needs.

The system is composed of individual modules that can be combined to create seating areas, workspaces, or storage solutions. Each module is designed to be intuitive, easy to use, and visually coherent.

Material selection plays a key role in the project, combining wood, textiles, and metal elements to achieve a balance between durability, comfort, and a warm aesthetic suitable for shared environments.
